Rails 001 Firststep

The first step

gem install rails
rails new webapp -B

Running first time

Install bundler && run server

cd webapp
bundler install
rails s -b 0.0.0.0 -p 3000 -e development

Generators

rails generate -h
rails generate controller Login

These files will be generated

    create  app/controllers/login_controller.rb #
    invoke  erb
    create    app/views/login
    invoke  test_unit
    create    test/controllers/login_controller_test.rb #
    invoke  helper
    create    app/helpers/login_helper.rb #
    invoke    test_unit
    invoke  assets
    invoke    coffee
    create      app/assets/javascripts/login.coffee #
    invoke    scss
    create      app/assets/stylesheets/login.scss #

Add new route and specify the LoginController handle this route

config/routes.rb

get '/login/index' # it will automatically find login controller and method index to serve this url
