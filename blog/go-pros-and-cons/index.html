Go Pros and Cons

languages

GO là gì

Go is an open source programming language that makes it easy to build simple, reliable, and efficient software.

Pros of GO

Go Is Fast

Go is compiled to machine code, it will naturally outperform languages that are interpreted or have virtual runtimes

Easy to Learn

Go’s syntax is small compared to other languages, and it’s easy to learn

Static Typing

Interface Types

Standard Library

Static Analysis Tools

Garbage Collection

Easier Concurrency Model

Cons of GO

  • No Generics
  • Interfaces Are Implicit
  • Poor Library Support
  • Difficult Community
  • Fractured Dependency Management

Whether to Use Go

If you’re working with bytes, Go might be a good choice. If you’re working with data, Go might not be a good choice.
